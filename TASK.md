# 综述：基于《杀戮尖塔2》的强化学习项目（pytorch）
搭建简易环境模拟游戏内容，不包含所有事件、药水、遗物、商店等等。
每层路线均只有：怪物（包含弱怪、普怪、精英、boss四种）+火堆（休息/升级一张牌）
战斗内不考虑所有的药水、遗物、事件的影响，只使用卡组对战怪物
强化学习模型分为战斗内/战斗外两个模型，战斗内模型负责使用卡组战胜遭遇的怪物，战斗外模型负责战胜怪物后的选牌以及火堆的抉择

# 任务清单：核心基于 decompiled/ 中的源码/反编译逻辑，实现游戏基础的战斗环境+强化学习训练环境，先完成战斗内模型
1.分析游戏中的所有卡牌，包含5个职业卡牌，无色牌，先古之民给予的牌，以及状态、诅咒、衍生牌等等
2.分析游戏中的所有敌人，包含一个四个场景
3.分析游戏战斗的所有机制，搭好完整战斗内环境，用于战斗内 PPO / MaskablePPO 训练
4.完成战斗内模型的训练/测试代码。战斗内模型的目标：对于不用的卡组+不同的怪物都能完成对战。
*核心问题：游戏怪物数量一定，机制很多却差异较大，模型是否需要识别每个怪物的ID从而应对不同怪物，是否需要提前了解怪物的机制or学习中领悟
5.支持以下训练/测试：
使用自定义卡组对战指定遭遇池的怪
使用随机卡组对战随机怪
未来可以在简易模拟路线中结合战斗外模型完成整个游戏流程
